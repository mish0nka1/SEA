European Football Tournaments 🏆

European Football Tournaments е ASP.NET Core MVC приложение, създадено за управление на футболни първенства, отбори и мачове. Проектът автоматизира изчисляването на футболната статистика и класирането въз основа на въведените резултати.
🚀 Функционалности

    Управление на мачове (CRUD): Пълна функционалност за добавяне, преглед, редакция и изтриване на мачове.

    Интуитивен дизайн: Модернизиран интерфейс с карти и сенки за по-добра подредба на информацията.

    Автоматично класиране: Системата пресмята точки, победи, загуби и голова разлика в реално време.

    Валидация на данни: Вградена проверка за задължителни полета и правилни релации между обектите.

🛠 Технологичен стек

    Backend: C# / ASP.NET Core 8.0 MVC

    Database: MS SQL Server / Entity Framework Core

    Frontend: Razor Pages, HTML5, CSS3, Bootstrap 5

⚙️ Инсталация и стартиране

    Клонирайте хранилището:
    Bash

    git clone https://github.com/потребител/EuropeanFootballTournaments.git

    Обновете връзката към базата данни в appsettings.json.

    Изпълнете миграциите в Package Manager Console:
    PowerShell

    Update-Database

    Стартирайте приложението чрез Visual Studio (F5).

🛡 Справяне с технически проблеми

По време на разработката са приложени фиксове за:

    NullReferenceException: Гарантирано зареждане на данни в падащите менюта при Create и Edit.

    Database Constraints: Оптимизирани Foreign Key връзки за избягване на конфликти при запис.
